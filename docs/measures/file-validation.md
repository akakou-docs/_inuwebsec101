## ファイルのバリデーションチェック

攻撃者は不正なファイルをアップロードしてくる場合があります。  
そこでアップロードされたファイルをバリデーションチェックする必要があります。

## チェックすべき項目

最低限チェックすべき項目は以下の通りでです。

- ファイル種類（JPG、PNG）
- ファイルのサイズ
