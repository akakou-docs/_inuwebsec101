# 危険なパラメータから守る

前提ページ：[HTTP メソッドとパラメータ](../../webapp/http-method/)  
推奨ページ：なし

Web アプリケーションは多くのパラメータを扱うわけですが、そのパラメータに悪意のあるデータを含ませることで、攻撃者は攻撃を行います。

この章では、危険なパラメータから Web ページを守る手段について説明します。
